# CI/CD Demo

This is a sample project demonstrating continuous integration and continuous deployment practices.

## Getting Started

Clone the repository and follow the setup instructions below.

### Prerequisites

- Git
- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Running Tests

```bash
npm test
```

### Deployment

```bash
npm run deploy
```
